#### 1. Install dependencies
```Shell

```

```Shell
npm i -g nodemon
npm run dev
```

homolog

Production
------
#### 2. Run project in production
```Shell
npm start
```


> **Note:**
> - If you use Windows, open prompt in admin mode
> - If you use the Linux Kernel, run the commands with the sudo prefix
